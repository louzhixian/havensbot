# Docs

## Purpose
Central documentation hub for the repo, including feature tracking, templates, and setup history.

## Key Files
- `docs/ARCHITECTURE.md`: system architecture overview (架构设计文档).
- `docs/features.md`: feature tracking and status history.
- `docs/setup_log.md`: setup and environment history.
- `docs/templates/feature_template.md`: feature entry template.
- `docs/templates/changelog_template.md`: Change Log template.
- `docs/templates/directory_readme_template.md`: directory README template.
- `prompts/editorial.item_enrichment.prompt.md`: editorial item enrichment prompt.
- `prompts/editorial.daily_report.prompt.md`: editorial daily report prompt.
- `prompts/deeper.article_summary.prompt.md`: deep dive article summary prompt.
- `prompts/editorial.thread_assistant.prompt.md`: editorial thread assistant prompt.
- `prompts/editorial.translation.prompt.md`: editorial translation prompt.

## Conventions
- Follow `agent.md` for documentation rules and directory Change Logs.
- Use templates from `docs/templates/` when creating new docs.
- Update this README Change Log when any file in `docs/` changes.
- Record prompt changes in `DECISIONS.md` per `prompts/README.md`.
